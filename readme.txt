starting scripts
0 - edit .env file with tour datas
1 - create virtual environment with: python .m venv .venv
2 - activate virtual environment with: .venv\Scripts\Activate
3 - install all dependencies with: pip install -r requirements.txt
4 - run app with: flask run

now you can test the app from this endpoint: http://127.0.0.1:5000/chat-stream/hello
